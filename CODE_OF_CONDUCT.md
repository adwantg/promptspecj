# Code of Conduct

This project follows the Contributor Covenant, version 2.1.

## Our standards

- Be respectful in technical disagreement.
- Assume good faith and focus on the code, tests, and user impact.
- Do not harass, insult, or exclude contributors.

## Enforcement

Report issues to `maintainers@promptspecj.io`. Project maintainers may remove content, restrict access, or ban participants who violate this code of conduct.
